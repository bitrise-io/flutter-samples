# flutter-ios-android-web

A Flutter sample app targeting iOS, Android, and Web platforms with Flutter version pinned via `.tool-versions` (asdf/mise).

## What it demonstrates

- Flutter app with `ios/`, `android/`, and `web/` directories all present
- Flutter version management via `.tool-versions`
- Widget test with flutter_test

## How to run

```bash
# Optional: install asdf or mise and run `asdf install` / `mise install`
# to pin Flutter to the version declared in .tool-versions

flutter pub get
flutter test
```
